VScode 단축키

- Command + N : 새파일
- Command + S : 저장
- Command + O : 열기
- Command + / : 주석

- Command + (+Shift) F : 찾기 (전체찾기)
- Command + , : 설정창
- Command + +, - : 확대, 축소

- Command + B : 사이드바 표시
- Command + P : 명령어 팔레트

- Command + D : 같은 폼 한번에 수정
- Option + click : 부분 수정
- Option + 방향키 : 이동
- Option + Shift + 방향키 : 위아래로 복붙

- Option + Shift + i : 한번에 수정

- Option + Shift + Drag : 줄에 맞춰서 보여줌s

[ html ]
- Command + Shift + V : 웹 미러링

[ 터미널 ]
- control + shift + ` : 새터미널
